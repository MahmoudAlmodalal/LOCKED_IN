# Task Management & Productivity System

A web-based task and productivity management system designed to help users organize tasks, build habits, and improve focus and efficiency.  
The platform provides full authentication, advanced task control, privacy features, and detailed productivity reports generated as PDF files.

---

## Overview
This system allows users to manage daily tasks, track habits, and monitor their productivity using a structured and secure workflow.  
It combines task management, habit tracking, and a Pomodoro-based focus system into a single, easy-to-use platform.

---

## Features

### Authentication & Security
- User registration and login  
- Secure authentication system  
- Protected routes and user-specific data  
- Password-protected (locked) tasks  

### Task Management
- Create, edit, and delete tasks  
- Task status tracking (pending, completed)  
- Task prioritization  
- Task sorting and filtering  
- Search functionality  

### Habit Tracking
- Create and manage daily habits  
- Track habit consistency and progress  
- Habit performance analytics  

### Productivity Tools
- Pomodoro-based productivity timer  
- Focus session tracking  
- Productivity insights  

### Reports & Analytics
- Generate PDF reports for:
  - Tasks
  - Habits
  - Achievements and progress
- Downloadable and shareable reports  

### User Experience
- Responsive design for all screen sizes  
- Clean and intuitive user interface  
- Fast and optimized frontend performance
